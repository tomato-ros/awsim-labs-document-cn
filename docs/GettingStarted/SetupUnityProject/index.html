    设置 Unity 项目#

    信息

    建议查看 快速入门演示 教程 在阅读本节之前.

    本页是设置 AWSIM Unity 项目的教程。

    环境准备#

    系统设置#

    1. 确保您的机器符合 所需的硬件规格.
      • 注意: PC 要求可能因仿真内容而异，这些内容可能会随着仿真器的发展而变化
    2. 准备一台 Ubuntu 22.04 安装.
    3. 安装 Nvidia drivers and Vulkan Graphics API.
    4. 安装 git.
    5. 跟随 DDS 配置指南.
    1. 确保您的机器符合 所需的硬件规格.
      • 注意: PC 要求可能因仿真内容而异，这些内容可能会随着仿真器的发展而变化
    2. 准备一台 Windows 10 or 11 (64 位) 安装.
    3. 安装 git.
    4. 安装 Microsoft Visual C++ Redistributable packages for Visual Studio 2015, 2017, 2019, and 2022 (X64 架构)

    ROS 2#

    AWSIM 附带一个 standalone 风味 Ros2ForUnity. 这意味着, 避免不同 ROS 2 版本之间的内部冲突, 您不应使用 ROS 2 来源运行 Editor 或 AWSIM 二进制文件.

    警告

    不要运行源自 ROS 2 的 AWSIM、Unity Hub 或 Editor.

    • 确保您用于运行 Unity Hub、Editor 或 AWSIM 的终端没有 ROS 2 来源.
    • 使用 ~/.bashrc 或 ~/.profile. 确保它不会遮挡您的工作环境:
      • 从 Ubuntu GUI 菜单运行 Unity Hub 需要从 ~/.profile.
      • 从终端运行 Unity Hub 将使用 ~/.profile 和 ~/.bashrc.
      • 从 UnityHub 运行 Unity Editor 会继承 Unity Hub 的环境设置.
    • 确保您的 Windows 环境变量是 ROS 2 自由的.

    Unity 安装#

    信息

    AWSIM 的 Unity 版本目前为 2022.3.36f1

    请按照以下步骤在您的计算机上安装 Unity:

    安装 UnityHub:#

    1. 安装 UnityHub 以管理 Unity 项目.请前往 Unity 下载页面 并下载最新版本 UnityHub.AppImage.
    2. 安装 Unity 2022.3.36f1 via UnityHub:
      • 打开新终端，导航到其中的目录 UnityHub.AppImage 是下载并执行以下命令: 
        ./UnityHub.AppImage

    安装 Unity Editor:#

    1. 点击 Installs 从左侧菜单中，然后单击 Install Editor 右上角的按钮.
    2. 在新窗口中，单击 Official releases 选项卡并检查 Unity 的当前版本. - 如果您找不到版本，请单击 Archive ，然后在 Unity 存档页面中搜索版本. 它会引导您 Unity 存档.

    3. 找到版本后，单击下载/安装按钮开始安装过程. - 此时，您的 Unity 安装过程应该已启动.

      === "Ubuntu 22" - *注意: 如果单击下载/安装按钮后安装过程仍未开始，请复制超链接(右键单击按钮并选择 Copy link address) 并将其添加为 Unity Hub 应用程序的参数。示例命令: 

      ./UnityHub.AppImage unityhub://2022.3.21f1/d91830b65d9b

    4. 安装成功后，该版本将在 Installs Unity Hub 中的选项卡.

    打开 AWSIM 项目#

    在 Unity 编辑器中打开 Unity AWSIM 项目:

    1. 确保您已克隆 AWSIM 存储库，并且 ROS 2 不是来源. 

      git clone git@github.com:autowarefoundation/AWSIM.git

    2. 启动 UnityHub. 

      ./UnityHub.AppImage

      信息

      如果要从 Ubuntu 应用程序菜单(不带终端)启动 Unity Hub，请确保已设置系统优化。可以肯定的是，在运行 Unity Hub 之前，至少运行一次终端。这将应用 OS 设置。

    3. 在 UnityHub 中打开项目

      • 点击 Projects 选项卡中的 Open 按钮,
      • 导航 AWSIM 存储库克隆到的目录，然后选择该目录,
      • 该项目应添加到 Unity Hub 的 Projects 选项卡中。要在 Unity 编辑器中启动项目，只需单击 AWSIM 项目.
      • 该项目现在可供使用.
      • 可以通过双击项目名称或从列表中选择它，然后单击右上角的 Open 按钮来打开项目.

    1. 输入 AWSIM 目录(确保 ROS 2 未成为源). 

      cd AWSIM

    2. 如果 Unity Editor 位于默认位置，请使用 editor 命令运行项目. 

      ~/Unity/Hub/Editor/[your_editor_version]/Editor/Unity -projectPath .

      信息

      请在路径中指定您的 Unity Editor 版本. 如果您的 Unity Editor 安装在不同的位置，请相应地调整路径.

    警告

    如果在启动 UnityEditor 时出现安全模式对话框，则可能需要安装 openssl.

    1. 下载 libssl
      $ wget http://security.ubuntu.com/ubuntu/pool/main/o/openssl1.0/libssl1.0.0_1.0.2n-1ubuntu5.13_amd64.deb
    2. 安装
      sudo dpkg -i libssl1.0.0_1.0.2n-1ubuntu5.13_amd64.deb

    导入扩展包#

    要在 Unity 中正确运行和使用 AWSIM 项目，需要下载存储库中未包含的地图包.

    1. 下载和导入 Nishishinjuku_URP_v0.2.0.unitypackage

      下载地图包

    2. 在 Unity Editor 中，从顶部的菜单栏中选择 Assets -> Import Package -> Custom Package... 并导航 Nishishinjuku_URP.unitypackage 您已下载并打开的文件.

    3. 点击 Import 按钮以导入包.
    4. Nishishinjuku 包应该在 Assets/AWSIM/Externals/ 目录.您可以从 Unity Editor 中的 Project 窗口访问该目录.

    信息

    Externals 目录将添加到 .gitignore 因为映射的文件大小很大，不应直接上传到存储库.

    导入图形资产#

    在 Editor 中运行演示#

    以下步骤介绍如何在 Unity Editor 中运行 demo:

    1. 打开位于 Assets/AWSIM/Scenes/Main 目录下的 AutowareSimulation.unity 场景
    2. 通过单击位于 Editor 顶部的 Play 按钮来运行模拟.
    3. 现在，您应该看到模拟在 Editor 中运行.

    如果您在运行模拟时遇到任何问题，请参阅 故障排除 部分或在 AWSIM Labs GitHub.
