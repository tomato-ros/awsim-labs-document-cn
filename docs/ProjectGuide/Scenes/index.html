    Scenes

    在 AWSIM Unity 项目中，有一个主场景 (AutowareSimulation) 和几个在开发过程中可能有用的其他场景。 本节介绍项目中每个场景的用途。

    AutowareSimulation#

    AutowareSimulation 场景是主场景，旨在与 Autoware 一起运行 AWSIM 模拟。 它允许轻松作，只需运行这个场景，使用正确的地图文件运行 Autoware，一切都应该开箱即用。

    PointCloudMapping#

    PointCloudMapping 是一个旨在使用 Unity 世界创建点云的场景。 使用 RGLUnityPlugin 和预制件“环境”(上面有带有 'Meshes' 的模型——我们能够获得模拟世界的 *.pcd 文件。

    SensorConfig#

    开发场景 SensorConfig 是为了对添加到 EgoVehicle 预制件中的传感器执行快速测试。 将 Lexus 预设件替换为您开发的车辆预设件，并检查应发布的所有数据是否都存在，是否与适当的主题有关，以及数据是否正确.

    NPCVehicleSample#

    开发 NPCVehicleSample 是为了对开发的车辆进行快速测试。 将出租车预制件替换为您开发的车辆预制件( EgoVehicle 或 NPCVehicle)，并检查基本配置是否正确。 关于如何开发自己的车辆并将其添加到项目中的描述在这个部分.

    NPCPedestrianSample#

    开发 NPCPedestrianSample 是为了对开发的行人进行快速测试。 将 NPC Pedestrian Test(NPC 行人测试)脚本中的 NPC prefab 替换为您开发的 prefab，并检查基本内容是否配置正确.

    TrafficIntersectionSample#

    开发 TrafficIntersectionSample 是为了对开发的交通交叉路口进行快速测试。 将 intersection 配置替换为您自己的配置，并检查它是否正常工作。 您可以添加其他灯光组并创建更大、更复杂的序列。 有关如何配置您自己的交通交叉路口的描述在此 部分.

    TrafficLightSample#

    开发 TrafficLightSample 是为了与控制它的脚本合作，对开发的交通信号灯模型进行快速测试。 用您自己的灯和配置替换，并检查它是否正常工作.

    RandomTrafficYielding#

    开发 RandomTrafficYielding 是为了在单个十字路口对开发的让行规则进行测试.

    RandomTrafficYieldingBirdEye#

    开发 RandomTrafficYielding 是为了对开发的让行规则进行测试，其中多个车辆在整个环境中移动.

    RGL 测试场景#

    下面描述的场景用于与外部库 RGLUnityPlugin (RGL) 相关的测试 - 您可以在此中阅读更多相关信息 部分.

    LidarSceneDevelop#

    场景 LidarSceneDevelop 可以用作如何设置 RGL 的完整、简约示例。 它包含 RGLSceneManager 组件、四个激光雷达和一个由地板和墙壁组成的环境.

    LidarSkinnedStress#

    场景 LidarSkinnedStress 可用于测试 RGL 的性能。 例如，与 “蒙皮网格” 相比，使用 “常规网格” 时性能会受到怎样的影响。 该场景包含大量动画模型，这些模型需要每帧更新网格，因此需要更多资源(CPU 和与 GPU 的数据交换).

    LidarDisablingTest#

    场景 LidarDisablingTest 可用于测试具有相似对象但具有不同配置的 'RGL' 性能。 它允许您检查当可以作为“网格”源的各种组件(“碰撞体”、“常规网格”、“蒙皮网格”等)被禁用时，“RGL”是否正常工作.

    LidarInstanceSegmentationDemo#

    LidarInstanceSegmentationDemo 是实例分段功能的演示场景。它包含一组分配了 ID 的 GameObjects 和将输出发布到 ROS2 主题的示例激光雷达。对 *GameObject * 进行分组以提供分配 ID 的不同方法。

    运行 demo scene:

    1. 打开场景: Assets/AWSIM/Scenes/Samples/LidarInstanceSegmentationDemo.unity
    2. 运行模拟
    3. 打开 rviz2
    4. 按如下方式设置 rviz2:
      • 固定 Frame: world,
      • PointCloud2 主题: lidar/instance_id,
      • 主题 QoS 如上屏幕所示。
      • 频道名称: enitity_id,
      • 为了更好地可视化，请禁用 Autocompute intensity 并将 min 设置为 '0' ，将 max 设置为 '50'.
