# awsim-labs-document-cn

- [英文文档(原版)](https://autowarefoundation.github.io/AWSIM-Labs/main/)
- [中文文档(当前)](https://tomato-ros.github.io/awsim-labs-document-cn/)

基于 [AWSIM-Labs](https://github.com/autowarefoundation/AWSIM-Labs) 进行翻译。

AWSIM Labs Documentation 中文文档项目，是面向自动驾驶开发者的AWSIM Labs 开源仿真器中文官方文档工程，由社区驱动完成 AWSIM Labs 英文原版文档的本土化翻译、适配与补充，旨在降低国内开发者使用这款基于 Unity 引擎的自动驾驶仿真工具的学习门槛，让中文社区更高效地掌握其部署、使用与二次开发方法。

AWSIM Labs 是 Autoware Foundation 基于 Tier IV 原版 AWSIM 打造的分支项目，作为 Autoware 生态核心的开源自动驾驶仿真器，它依托 Unity 构建了高还原度的虚拟仿真环境，支持 ROS 2 原生通信，可实现车辆控制、多传感器模拟、复杂交通场景仿真等核心能力，是自动驾驶算法训练、功能验证、全链路测试的重要工具。

本中文文档项目将完整覆盖 AWSIM Labs 的核心内容，从基础的环境搭建、快速上手，到进阶的传感器配置、场景定制，再到深度的二次开发、性能优化，形成体系化、易理解的中文指引，同时同步官方文档的更新迭代，保证内容的时效性与准确性。

本项目的核心价值在于打通语言壁垒，赋能中文开发者生态：无论是自动驾驶领域的入门学习者、高校科研人员，还是企业开发工程师，都能通过这份中文文档快速掌握 AWSIM Labs 的使用技巧，减少因英文文档带来的学习成本；同时，文档将保留开源协作特性，欢迎社区开发者参与内容完善、问题反馈与案例补充，让文档在实践中持续优化，成为国内 AWSIM Labs 使用者的必备参考手册。

目前，AWSIM Labs Documentation 中文文档已完成核心模块的梳理与翻译，涵盖项目介绍、快速开始、环境配置、核心功能使用等基础内容，后续将持续更新高级用法、实战案例、常见问题排查等板块，助力更多中文开发者基于 AWSIM Labs 开展自动驾驶仿真研发工作。

## 关于作者

公众号《番茄ROS机器人》作者，长期致力于 ROS2 与 Autoware 框架下的低速无人驾驶解决方案研发，专注技术实战与经验沉淀，分享低速无人系统开发中的技术难点、解决方案与行业思考，与开发者共同推动低速无人驾驶技术的落地与创新。
